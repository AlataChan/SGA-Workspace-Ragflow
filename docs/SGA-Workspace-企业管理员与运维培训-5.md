
### B-M0 首次登录（3–4 页）

#### USER-01 拿到账号后的第一步

**本页目标**：用户能成功登录并进入工作台。

**页内要点**：
- 管理员已经为你创建好账号，并告知你：① 系统地址 ② 用户名 ③ 初始密码
- 打开浏览器，输入系统地址
- 在登录页面输入用户名和密码，点击登录
- 首次登录后会进入工作台（`/workspace`）——这就是你的日常工作界面

**演示步骤**：
1. 打开浏览器，输入系统地址（例如：`http://ai.company.com`）
2. 在登录页面的"用户名"框输入账号
3. 在"密码"框输入初始密码
4. 点击"登录"
5. 看到工作台界面，说明登录成功

**如果登录失败**：

| 问题 | 可能原因 | 解决方式 |
|------|---------|---------|
| 提示"用户名或密码错误" | 密码输错了 | 检查大小写，确认是否用了中文输入法 |
| 提示"账号已停用" | 账号被管理员停用 | 联系管理员恢复账号 |
| 页面加载不出来 | 网络问题或地址错误 | 确认网络可用、地址是否正确 |

### B-M1 认识工作台（4–6 页）

#### USER-02 工作台界面介绍

**本页目标**：用户能认出工作台各个区域的功能。

**工作台主要区域**：

```
┌─────────────────────────────────────────────────┐
│ 顶部导航：公司 Logo、用户名、设置入口            │
├──────────┬──────────────────────────────────────┤
│ 左侧边栏 │                                      │
│          │                                      │
│ - AI助手 │          右侧主区域                  │
│   列表   │       （对话区域）                   │
│   (按部  │                                      │
│   门分组)│                                      │
│          │                                      │
│ - 知识   │                                      │
│   图谱   │                                      │
└──────────┴──────────────────────────────────────┘
```

**页内要点**：
- **左侧**：你有权限的所有 AI 助手（按部门分组显示）
- **右侧**：当前对话的聊天界面
- **左侧底部**：知识图谱入口（如果管理员已授权给你）
- **如果左侧是空的**：说明还没有授权给你任何助手，联系管理员

---

#### USER-03 找到并选择 AI 助手

**本页目标**：用户能找到自己想用的 AI 助手并开始对话。

**页内要点**：
- 左侧列表按**部门分组**展示所有你有权限的助手
- 每个助手显示：头像、名字、职位描述、在线状态（绿点=可用）
- 如果助手旁边是灰色/红色点，说明该助手暂时不可用，联系管理员

**演示步骤**：
1. 在左侧找到目标助手（例如"销售分析助手"）
2. 点击助手卡片
3. 右侧出现对话界面，底部有输入框
4. 在输入框里输入你的问题
5. 按回车或点击发送按钮
6. 等待助手回复

---

#### USER-04 发送消息与附件

**本页目标**：用户能发送文字消息，以及上传文件给 AI 助手处理。

**页内要点**：
- **发送文字**：在输入框输入内容，按回车或点发送图标
- **上传文件**（如果该助手支持）：
  - 点击输入框左侧的回形针图标
  - 选择文件（支持 PDF、Word、TXT、图片等）
  - 文件上传后会显示文件名，然后再发送问题
- **Shift+回车**：换行（不发送）
- **回车**：直接发送

**注意（讲师口述）**：
> 不是所有助手都支持上传文件——这取决于背后连接的 AI 平台能否处理文件。如果上传后没有反应或报错，该助手可能不支持文件输入。

---

#### USER-05 查看和管理历史对话

**本页目标**：用户能找到过去的对话记录，并开始新对话。

**页内要点**：
- 每次与某个助手对话都会生成一个"会话"
- 历史会话显示在左侧面板里（点击助手后展开）
- 点击历史会话：继续之前的对话
- 点击"新对话"按钮：开始一个全新的空白对话
- 历史对话在服务器上保存，换设备登录也能看到

**演示步骤**：
1. 与助手进行一次对话（发 2-3 条消息）
2. 点击"新对话"，输入框清空，开始新会话
3. 在历史记录里点击第一次的对话，看到原来的消息

### B-M2 使用知识图谱（4–6 页）

#### USER-06 知识图谱是什么，能做什么

**本页目标**：用户理解知识图谱的作用，知道什么时候用它。

**页内要点**：
- 知识图谱 = 公司把自己的知识"喂给"AI 的方式
- 当你在工作台开启知识图谱，AI 回答问题时会优先从公司资料里找答案
- 适合用知识图谱的场景：
  - 查产品规格、操作手册
  - 了解公司政策、流程文档
  - 基于公司内部资料的问答
- 如果你在左侧看不到知识图谱，说明管理员还没有授权给你，联系管理员即可

---

#### USER-07 在对话中使用知识图谱

**本页目标**：用户能在与 AI 助手对话时激活知识图谱。

**页内要点**：
- 在与 AI 助手对话的界面里，找到"知识图谱"或"知识库"开关/按钮
- 开启后，发出的问题会结合知识图谱的内容来回答
- AI 的回答会标注来源，点击来源可以查看原始文档的片段

**演示步骤**：
1. 选择一个 AI 助手，进入对话界面
2. 找到知识图谱开关并开启（选择"产品手册知识库"）
3. 输入问题：`XXX 产品的最大承载重量是多少？`
4. 观察 AI 回复中的来源标注
5. 点击来源，查看原始文档片段

### B-M3 实用技巧（3–4 页）

#### USER-08 常用操作速查

**本页目标**：用户掌握日常使用中最常用的快捷操作。

| 操作 | 方式 |
|------|------|
| 开始新对话 | 点击"新对话"按钮 |
| 回到某次历史对话 | 点击左侧历史记录 |
| 上传文件 | 输入框旁的回形针图标 |
| 换行（不发送） | Shift + 回车 |
| 发送消息 | 回车 / 点击发送按钮 |
| 复制 AI 的回复 | 鼠标选中文字 → 右键复制 / 或点击回复旁的复制图标 |
| 切换深色/浅色主题 | 右上角用户图标 → 设置 → 主题 |

---

#### USER-09 遇到问题时怎么办

**页内要点**：

| 问题描述 | 自己先试 | 如果还不行 |
|---------|---------|-----------|
| AI 助手不回复 / 转圈 | 等待 10–30 秒，有时 AI 需要时间思考 | 刷新页面，如果仍然没有，联系管理员检查助手状态 |
| 找不到某个助手 | 确认在正确的部门分组里找 | 联系管理员确认是否有授权 |
| 上传文件失败 | 检查文件大小（不超过 100MB）和格式 | 换一个文件格式再试 |
| 登录后页面空白 | 强制刷新（Ctrl+Shift+R） | 清除浏览器缓存，重新登录 |
| 忘记密码 | 无法自助重置 | 联系管理员重置密码 |
